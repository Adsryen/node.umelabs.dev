浏览器的选择

关于浏览器的选择，在YouTube频道也有做视频，有关于浏览器配置和安全方面的，还有Tor浏览器有关的视频。但是，通过谷歌分析发现还有一些人使用并不是很安全的浏览器。浏览器的选择或者说软件的选择上有几个需要注意的点，下面我们来看看。

前言

在开始之前可以看看这篇关于UC浏览器安全问题的文章。

背景和题外话

背景

为什么想要写这样一篇文章呢？主要是因为看到谷歌分析的数据而想到要写这样一篇文章。因为一些人使用的浏览器让人感到很忧虑。

题外话

在说浏览器之前，这里说一下选择软件或者服务的一些需要注意的事项。

  1. 尽量选择非商业公司的软件或者服务
  2. 优先选择开源的软件或者服务
  3. 查看他们是否收集私人信息，如个人电话号码，邮箱，出生年月等等
  4. 查看他们是否把你的信息共享给其他公司或者机构
    其中开源是最重要的，因为开源大家可以看到源代码，知道他们做了什么，当然有的公司可能只开源了前端代码而没有开源后端代码。

浏览器选择

桌面端或者说PC/Desktop版

因为前面的题外话，我们可以直接排除一些所谓“国产”浏览器，包括腾讯，搜狗，阿里巴巴等商业公司的浏览器产品。
以下是一个推荐排行：

  1. Tor浏览器或者叫洋葱路由
  2. FireFox浏览器又叫火狐浏览器（⚠️注意不要下载中国特供版）
  3. Chromium (Chrome浏览器或谷歌浏览器开源版本)
  4. Brave 浏览器
  5. Opera 浏览器 (桌面版自带VPN，注意不要下载中国特供版)
  6. Safari (仅支持苹果设备，也有Windows版本，不过停止更新了)
  7. Edge 浏览器（这里说的是基于Chromium开发的版本）
  8. Chrome浏览器或谷歌浏览器（说实话不是很喜欢，但的确是个好的浏览器）

总结

选择开源的浏览器是最好的，优先推荐Tor，再然后是FireFox。如果你有选择困难症，可以看看FireFox写的这篇文章对比。
